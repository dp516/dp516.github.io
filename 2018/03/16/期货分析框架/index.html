期货基本面量化分析框架演示

发表于 2018-03-16 | 分类于 金融投资

怎样才能做好期货基本面分析？对现货行业市场、宏观环境、产业链头部企业三者均要做足功课，形成自己的市场驱动逻辑，同时要面对海量的数据信息，进行筛选和甄别，来支持、验证和改善自己的逻辑，需要大量的时间和精力。驱动市场价格的逻辑在不断变化，为了节约信息处理方面的时间，需要建立基本面量化框架。

【使用说明】

  • 本框架采用 python + jupyter notebook + Wind的python数据接口 编制并导出网页而来。
  • 目的：实时跟踪品种基本面和相关市场的数据，根据基本面或技术面等数据建立价格预期模型
  • 数据来源：Wind、文华财经、tushare、收集整理的Excel文件等，不限
  • 特点：将数据展示为动态图表，互动性强、后期扩展性强
