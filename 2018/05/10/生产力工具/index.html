我的生产力工具

发表于 2018-05-10 | 分类于 IT应用

有许多软件或app能够帮助我们提高工作效率和生活品质，笔者通过平时的使用心得，经过精挑细选，从上网、办公、编程、图像视频处理等方面总结了相应的一些软件。分享如下：

名称 性质 功能 特色
Firefox（V10） 软件 浏览及管理网络信息、调试代码 速度快、功能强大
finaldata 软件 恢复硬盘上删除的文件 方便快速，尤其适合office文档恢复
FileLocatorPro 软件 文件文本内容搜寻 批量对PDF文件的内容搜索，速度快
MouseWithoutBordersSetup 软件 局域网键盘鼠标控制操作多个电脑 多电脑操作神器
TC 软件 类似键盘精灵，可打包成exe 可编程，速度快
HA-IcoFX1.5-lifu 软件 ico制作 小巧方便
fscapture 软件 截图 小巧，滚动截屏，生成PDF
FileZilla 软件 FTP上传下载 小巧方便
Anaconda3 软件 python集成软件包 软件包管理方便
Jupyter Notebook 软件 数据分析 python数据分析神器
VS Code 软件 编程（适合js\python等） 速度快、插件丰富、集成终端
AVS Video Editor 软件 视频编辑器 做带视频的电子相册
Calibre Portable 软件 电子书管理 电子书制作，kindle电子书上传
cherrytree 软件 文本编辑器 写日记，做笔记，离线知识分类存档管理
Xmind 软件 思维导图编辑器 做思维导图，整理思路
滴答清单 web、手机app 时间管理 免费功能基本能满足手机和PC端同步实践GTD理念
Typora 软件 Markdown写作 区别于传统，人性化，细节到位，如excel表格可直接粘贴过来
